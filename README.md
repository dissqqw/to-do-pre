# To-do App — Менеджер задач

Простое и удобное приложение для управления списком дел с сохранением данных в браузере.

##  Функциональность

-  **Просмотр задач** — при загрузке отображаются предустановленные задачи
-  **Добавление** — новая задача добавляется в начало списка
-  **Удаление** — любая задача может быть удалена
-  **Копирование** — создание копии задачи на первом месте
-  **Редактирование** — изменение текста задачи прямо в списке
-  **Сохранение** — все задачи сохраняются в localStorage и не пропадают после перезагрузки

##  Технологии

- **HTML5** — структура приложения (шаблон для задач)
- **CSS3** — стили (уже готовы в проекте)
- **JavaScript (ES6+)** — вся логика работы
- **Git** — система контроля версий
